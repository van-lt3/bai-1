1. Giải thích các câu lệnh: git init, git add <file_name>, git add ., git commit -m "message"

- git init: Dùng để khởi tạo một Git repository mới trong folder hiện tại. Sau khi chạy, Git sẽ tạo thư mục ẩn .git để lưu trữ toàn bộ lịch sử và cấu hình của repo.Khởi tạo các vùng: Working directory, Staging area, Repository

- git add <file_name>: Đưa file chỉ định vào Staging Area (vùng chuẩn bị commit).

- git add .: Đưa tất cả file mới hoặc file đã thay đổi trong thư mục hiện tại vào Staging Area.
 
- git commit -m "message": Ghi lại (snapshot) trạng thái của toàn bộ các file trong staging area vào repository, kèm theo lời nhắn (commit message) để mô tả thay đổi.
